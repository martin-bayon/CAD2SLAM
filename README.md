# CAD2SLAM

**CAD2SLAM** is a framework for adaptive projection between CAD blueprints and SLAM maps.  
The project provides tools to align, project, and compare architectural CAD information with robot-generated maps, supporting the integration of prior structural knowledge into robotic mapping pipelines.

This repository contains the implementation associated with the paper: "**CAD2SLAM: Adaptive Projection Between CAD Blueprints and SLAM Maps**," *IEEE Robotics and Automation Letters, 2025*.
[https://ieeexplore.ieee.org/document/10816387](https://ieeexplore.ieee.org/document/10816387)
